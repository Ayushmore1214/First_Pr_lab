## [1.0.0]  -  2024-01-01

### Added
- Initial project launch
- DevOps Glossary webpage with 20 terms across 5 categories
- Dark mode toggle with system preference detection
- Real-time search filtering by term text
- Category filter buttons (All, Core Concepts, Containers, Kubernetes, CI/CD, Observability)
- Keyboard shortcut: Press `/` to focus the search bar, `Esc` to clear
- Fully responsive layout (mobile, tablet, desktop)
- Two-stage contributing system (intro PR + real issues)
- GitHub Actions: DCO check, HTML validation, PR title enforcement, welcome bot
- GitHub issue templates: bug report, feature request, intro PR
- PR template
- Full CONTRIBUTING.md guide written for complete beginners
- CODE_OF_CONDUCT.md (Contributor Covenant)
- CONTRIBUTORS.md with contributors table
- MIT License
